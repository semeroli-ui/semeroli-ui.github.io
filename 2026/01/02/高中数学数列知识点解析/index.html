高中数学数列专项练习题（生活应用版）

高中数学数列专项练习（全题型版）

本练习包含基础概念辨析、生活应用选择题、综合简答题、递推压轴题以及高考真题，旨在全方位检验数列知识掌握情况。

一、基础概念（概念理解类）

1. 【情境：理财计划与阶梯电价】

请根据以下生活场景，判断对应的数列类型并写出其核心特征：

场景 A：某市实行阶梯电价，第一档电量内每度 0.5 元。若每多出一档，每度电的价格固定增加 0.1 元。

场景 B：某理财产品号称”资产翻倍计划”，即每一年的资产总额都是上一年的 1.1 倍。

【解析】：

场景 A：属于等差数列。特征是”从第二项起，每一项与前一项的差等于同一个常数（公差 $d=0.1$）”。其通项公式是线性函数型的。

场景 B：属于等比数列。特征是”从第二项起，每一项与前一项的比等于同一个常数（公比 $q=1.1$）”。其通项公式是指数函数型的。

【易错点提示】：
在判断等比数列时，必须确保首项 $a_1 \neq 0$ 且公比 $q \neq 0$。在生活场景中，若提到”增长率为 $10%$”，公比应为 $1+10% = 1.1$，而非 $0.1$。

二、选择题（生活应用类）

2. 【情境：短视频博主的激励计划】

某短视频平台为了激励创作者，推出”点赞翻倍金”。规定：创作者发布的某条视频，第一天的点赞量记为 $a_1$，此后每天的点赞量是前一天的 $2$ 倍。已知该视频在前 $4$ 天的总点赞量为 $3000$，则该视频第 $1$ 天的点赞量 $a_1$ 为（ ）

A. 100
B. 200
C. 300
D. 400

答案：B

解析：这是一个公比 $q=2$ 的等比数列。
根据求和公式 $S_n = \frac{a_1(1-q^n)}{1-q}$，代入已知条件：
$3000 = \frac{a_1(1-2^4)}{1-2} \Rightarrow 3000 = \frac{a_1(1-16)}{-1} \Rightarrow 3000 = 15a_1$。
解得 $a_1 = 200$。

易错点提示：容易混淆 $q^n$ 中的 $n$。此处前 $4$ 天即 $n=4$，不要习惯性地写成 $n-1$（那是通项公式）。

三、简答题（综合实践类）

3. 【情境：高考倒计时复习计划】

为了迎接高考，小强制定了英语单词复习计划。他在倒计时第 100 天时背诵了 20 个单词，以后每天比前一天多背 2 个单词。

（1）求小强在倒计时第 $n$ 天背诵单词的数量 $a_n$ 的通项公式。

（2）若小强坚持复习 30 天，这 30 天内他一共背诵了多少个单词？

解析：

（1）由题意可知，单词数构成等差数列，首项 $a_1 = 20$，公差 $d = 2$。
通项公式为：$a_n = a_1 + (n-1)d = 20 + (n-1) \times 2 = 2n + 18$。

（2）求前 30 天的总和 $S_{30}$。
$S_{30} = 30a_1 + \frac{30 \times (30-1)}{2}d$
$S_{30} = 30 \times 20 + \frac{30 \times 29}{2} \times 2 = 600 + 870 = 1470$（个）。

易错点提示：

  1. 计算 $S_n$ 时，公式 $\frac{n(a_1+a_n)}{2}$ 或 $na_1 + \frac{n(n-1)}{2}d$ 容易混淆分母或系数。
  2. 注意 $n$ 的取值范围应为 $n \in {1, 2, \dots, 30}$。

四、压轴题（逻辑推理与创新类）

4. 【情境：购房分期付款模型】

某学生毕业后贷款买房，贷款总额为 $L$ 元，月利率为 $r$。还款方式为”等额本息”，即每个月偿还固定的金额 $X$ 元。设第 $n$ 个月还款后剩余的欠款为 $a_n$（注意：$a_0 = L$）。

（1）写出 $a_n$ 与 $a_{n-1}$ 的递推关系式。

（2）证明数列 ${a_n - \frac{X}{r}}$ 是等比数列，并以此推导 $X$ 的表达式（设共需 $N$ 个月还清）。

解析：

（1）第 $n$ 个月的欠款 = 上月欠款 $\times (1+r) - $ 本月还款金额 $X$。
递推式：$a_n = (1+r)a_{n-1} - X$。

（2）构造等比数列：
$a_n - \frac{X}{r} = (1+r)a_{n-1} - X - \frac{X}{r} = (1+r)a_{n-1} - \frac{Xr+X}{r} = (1+r)(a_{n-1} - \frac{X}{r})$。

因此，${a_n - \frac{X}{r}}$ 是以 $a_0 - \frac{X}{r} = L - \frac{X}{r}$ 为首项，$1+r$ 为公比的等比数列。

其通项为：$a_n - \frac{X}{r} = (L - \frac{X}{r})(1+r)^n$。

当还清贷款时，$a_N = 0$，则：
$0 - \frac{X}{r} = (L - \frac{X}{r})(1+r)^N$
$\frac{X}{r} = (\frac{X}{r} - L)(1+r)^N \Rightarrow X = \frac{Lr(1+r)^N}{(1+r)^N - 1}$。

易错点提示：

  1. 符号陷阱：还款是减法，利息是加法，不要写反。
  2. 初始项：由于第 0 个月（刚贷款时）金额为 $L$，通常用 $a_0$ 开始推导更简洁，若用 $a_1$ 开始则需注意项数。

五、高考原题（实战演练类）

5. 【高考真题：2023年全国甲卷（理）第17题·改编】

已知数列 ${a_n}$ 的前 $n$ 项和为 $S_n$，且 $2S_n = 3a_n - 3$。

（1）求 ${a_n}$ 的通项公式。

（2）若 $b_n = \log_3 a_n$，求数列 ${ \frac{1}{b_n b_{n+1}} }$ 的前 $n$ 项和 $T_n$。

解析：

（1）当 $n=1$ 时，$2a_1 = 3a_1 - 3 \Rightarrow a_1 = 3$。

当 $n \ge 2$ 时，$2S_n = 3a_n - 3$，$2S_{n-1} = 3a_{n-1} - 3$。

两式相减：$2(S_n - S_{n-1}) = 3a_n - 3a_{n-1} \Rightarrow 2a_n = 3a_n - 3a_{n-1} \Rightarrow a_n = 3a_{n-1}$。

所以 ${a_n}$ 是首项为 $3$，公比为 $3$ 的等比数列。

通项公式 $a_n = 3 \cdot 3^{n-1} = 3^n$。

（2）$b_n = \log_3 3^n = n$。

则 $\frac{1}{b_n b_{n+1}} = \frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$（裂项相消法）。

$T_n = (1 - \frac{1}{2}) + (\frac{1}{2} - \frac{1}{3}) + \dots + (\frac{1}{n} - \frac{1}{n+1}) = 1 - \frac{1}{n+1} = \frac{n}{n+1}$。

易错点提示：

  1. $S_n$ 与 $a_n$ 的关系：使用 $S_n - S_{n-1} = a_n$ 时，必须强调 $n \ge 2$，并单独检验 $n=1$。
  2. 裂项相消：裂项后一定要检查中间项是否真的能抵消，以及首尾留下的项正负号是否正确。

六、补充练习题

1. 【情境：智能手表步数挑战】

小明参加了学校”每日步数阶梯挑战”。已知第一天要求走 6000 步，以后每天比前一天增加 500 步。若小明坚持完成了这项挑战，请问在第 12 天他需要走的步数是（ ）

A. 11000
B. 11500
C. 12000
D. 12500

答案：B

解析：步数构成首项 $a_1 = 6000$，公差 $d = 500$ 的等差数列。
根据等差数列通项公式 $a_n = a_1 + (n-1)d$：
$a_{12} = 6000 + (12 - 1) \times 500 = 6000 + 5500 = 11500$（步）。

易错点提示：容易误算为 $a_{12} = a_1 + 12d$，注意公式中是 $n-1$。

2. 【情境：手机电池寿命衰减】

某品牌手机电池的容量会随着充电次数的增加而衰减。假设每次完全充放电循环后，电池容量都会变为上一次循环后的 $99.8%$。已知新手机的初始电量为 $4500\text{ mAh}$，求该手机在经过 100 次循环后，电池容量的大致表达式为（ ）

A. $4500 \times 0.998^{99}$
B. $4500 \times 0.998^{100}$
C. $4500 \times (1 - 0.998^{100})$
D. $4500 + 100 \times (-0.002)$

答案：B

解析：电池容量构成公比 $q = 0.998$ 的等比数列。
初始（$n=0$ 次循环）为 $4500$。
第 1 次循环后：$a_1 = 4500 \times 0.998$；
第 2 次循环后：$a_2 = 4500 \times 0.998^2$；
…
第 100 次循环后：$a_{100} = 4500 \times 0.998^{100}$。

易错点提示：注意”第 $n$ 次循环后”对应的幂次。若把初始值看作 $a_1$，则第 100 次循环后是 $a_{101}$，幂次依然是 100。

本练习旨在通过真实生活情境，帮助同学们掌握等差数列、等比数列及其求和、递推的应用。

极限思想：压轴题往往考察对数列”长期趋势”的理解。

构造数列：$a_n = q a_{n-1} + k$ 型数列转化为等比数列时的计算符号。
