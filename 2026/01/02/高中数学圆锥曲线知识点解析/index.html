高中数学圆锥曲线专项练习（全题型版）

本练习涵盖圆锥曲线（椭圆、双曲线、抛物线）的核心考点，通过生活实景建模，解析几何性质与代数运算。

一、 基础概念（定义理解类）

  1. 【情境：园艺中的椭圆花坛】
    园艺师在公园绿地上设计一个椭圆花坛。他的做法是：找两根木桩钉在地面 $F_1, F_2$ 处，取一根长度为 $10$ 米的无弹性绳子，将绳子的两端分别系在两根木桩上。然后用一支木棍撑直绳子，使木棍尖端 $P$ 在地面上滑动。
    （1）若两木桩间的距离为 $6$ 米，求木棍尖端轨迹的方程（以线段 $F_1F_2$ 的中点为原点，所在直线为 $x$ 轴）。
    （2）若园艺师想让花坛更“圆”一些，他应该缩短还是拉长两木桩之间的距离？

【解析】：
（1）根据椭圆定义，轨迹上点 $P$ 到两定点 $F_1, F_2$ 的距离之和为定值 $2a$。
由题意，$2a = 10 \Rightarrow a = 5$；焦距 $2c = 6 \Rightarrow c = 3$。
因为 $b^2 = a^2 - c^2 = 5^2 - 3^2 = 16$，所以 $b = 4$。
轨迹方程为：$\frac{x^2}{25} + \frac{y^2}{16} = 1$。
（2）要使椭圆更圆，离心率 $e = \frac{c}{a}$ 应更小。由于绳长 $2a$ 固定，则需减小焦距 $2c$，即缩短两木桩间的距离。

【易错点提示】：

忽略定义中 $2a > 2c$ 的限制。若绳长等于木桩距离，轨迹是线段；若小于，则不构成轨迹。

离心率 $e$ 越接近 $0$，椭圆越圆；越接近 $1$，椭圆越扁。

二、 选择题（建筑美学类）

  1. 【情境：发电厂冷却塔的廓线】
    许多发电厂的冷却塔外形呈现为双曲线旋转而成的形状，这种设计能够增强结构的稳定性。已知某冷却塔的水平截面廓线（如图所示）是一条双曲线的一部分，其标准方程为 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 \ (a>0, b>0)$。已知该冷却塔最窄处的直径为 $20$ 米，其渐近线方程为 $y = \pm \sqrt{3}x$，则该双曲线的焦距为（ ）
    A. 20

B. $20\sqrt{3}$

C. 40

D. $40\sqrt{3}$

答案： C

解析： 1. 最窄处对应双曲线的实轴，直径为 $20$ 米，即 $2a = 20 \Rightarrow a = 10$。
2. 渐近线方程为 $y = \pm \frac{b}{a}x = \pm \sqrt{3}x \Rightarrow \frac{b}{a} = \sqrt{3}$。
3. 代入 $a=10$，得 $b = 10\sqrt{3}$。
4. 焦参数 $c^2 = a^2 + b^2 = 100 + 300 = 400 \Rightarrow c = 20$。
5. 焦距为 $2c = 40$。

易错点提示： 注意双曲线中 $a, b, c$ 的关系是 $c^2 = a^2 + b^2$，而椭圆中是 $a^2 = b^2 + c^2$，两者极易混淆。

三、 简答题（光学性质类）

  1. 【情境：手电筒与探照灯的反射原理】
    抛物线具有神奇的光学性质：从焦点发出的光线经抛物线反射后，反射光线平行于抛物线的对称轴。现有一探照灯，其反光镜面的纵截面是抛物线 $y^2 = 4x$ 的一部分。
    （1）已知灯泡的中心位于焦点 $F$ 处，请写出焦点 $F$ 的坐标和准线 $l$ 的方程。
    （2）若在探照灯反光镜面上有一点 $P$，其横坐标为 $3$，求点 $P$ 到灯泡中心 $F$ 的距离。

解析：
（1）抛物线方程为 $y^2 = 4x$，对应标准形式 $y^2 = 2px$，则 $2p = 4 \Rightarrow p = 2$。
焦点 $F(\frac{p}{2}, 0)$ 即 $F(1, 0)$。
准线方程 $x = -\frac{p}{2}$ 即 $x = -1$。
（2）利用抛物线的定义：抛物线上任意一点到焦点的距离等于它到准线的距离。
点 $P$ 的坐标为 $(3, y_P)$，到准线 $x = -1$ 的距离 $d = |3 - (-1)| = 4$。
因此，$|PF| = 4$。

易错点提示： 1. 抛物线的开口方向。若方程为 $x^2 = 2py$，则对称轴是 $y$ 轴。
2. 距离转换。直接用两点间距离公式涉及 $y$ 的计算较繁琐，利用定义转化为到准线的距离会非常简单。

四、 压轴题（综合探究类）

  1. 【情境：现代物流中的无人机配送】
    某物流中心（位于原点 $O$）正试图划定一个配送区域。已知一架无人机在运行过程中，到某信号源 $F(1, 0)$ 的距离与其到直线 $l: x = 2$ 的距离之比恒为 $\frac{\sqrt{2}}{2}$。
    （1）求无人机飞行轨迹 $C$ 的方程，并指出其形状。
    （2）设直线 $m: y = kx + 1$ 与轨迹 $C$ 交于 $A, B$ 两点，是否存在实数 $k$，使得以 $AB$ 为直径的圆过原点 $O$？若存在，求出 $k$ 的值；若不存在，说明理由。

解析：
（1）设轨迹上点为 $P(x, y)$。由题意：$\frac{\sqrt{(x-1)^2 + y^2}}{|x-2|} = \frac{\sqrt{2}}{2}$。
平方得：$4[(x-1)^2 + y^2] = 2(x-2)^2 \Rightarrow 4(x^2 - 2x + 1 + y^2) = 2(x^2 - 4x + 4)$。
化简：$2x^2 + 4y^2 = 4 \Rightarrow \frac{x^2}{2} + y^2 = 1$。
形状为椭圆。
（2）联立直线与椭圆：
$\begin{cases} y = kx + 1 \ x^2 + 2y^2 = 2 \end{cases} \Rightarrow x^2 + 2(kx+1)^2 = 2 \Rightarrow (1+2k^2)x^2 + 4kx = 0$。
解得 $x_1 = 0$（即原点或交点之一），$x_2 = -\frac{4k}{1+2k^2}$。
若以 $AB$ 为直径的圆过原点，则 $OA \perp OB \Rightarrow \vec{OA} \cdot \vec{OB} = 0$。
设 $A(0, 1)$（由 $x_1=0$ 代入直线方程得），$B(x_2, y_2)$。
$\vec{OA} \cdot \vec{OB} = 0 \cdot x_2 + 1 \cdot y_2 = 0 \Rightarrow y_2 = 0$。
将 $y_2 = 0$ 代入直线方程得 $0 = kx_2 + 1 \Rightarrow k(-\frac{4k}{1+2k^2}) + 1 = 0$。
化简得：$1+2k^2 - 4k^2 = 0 \Rightarrow 2k^2 = 1 \Rightarrow k = \pm \frac{\sqrt{2}}{2}$。
且判别式 $\Delta > 0$ 恒成立。故存在 $k = \pm \frac{\sqrt{2}}{2}$ 满足条件。

易错点提示： 1. 轨迹求法：利用圆锥曲线的统一性（离心率定义）可以快速判断形状。
2. 圆过原点转化：转化为向量点积为 0 或勾股定理，是解析几何中常见的几何条件代数化技巧。

五、 高考原题（实战演练类）

  1. 【高考真题：2024年全国甲卷（理）第10题·改编】
    已知椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \ (a>b>0)$ 的左、右焦点分别为 $F_1, F_2$，点 $P$ 在 $C$ 上。若 $\Delta PF_1F_2$ 是面积为 $b^2$ 的等腰三角形，则椭圆 $C$ 的离心率为（ ）
    A. $\frac{1}{2}$

B. $\frac{\sqrt{2}}{2}$

C. $\frac{\sqrt{3}}{2}$

D. $\frac{\sqrt{5}-1}{2}$

解析：

$\Delta PF_1F_2$ 以 $F_1F_2$ 为底边时，高 $h = y_P = b$。由于点 $P$ 在椭圆上，$y_P \le b$，所以 $P$ 必须为短轴端点。此时面积 $S = \frac{1}{2} \cdot 2c \cdot b = bc = b^2 \Rightarrow c = b$。

代入 $a^2 = b^2 + c^2 = 2c^2$，则 $a = \sqrt{2}c$。

离心率 $e = \frac{c}{a} = \frac{c}{\sqrt{2}c} = \frac{\sqrt{2}}{2}$。

若以 $PF_1$ 为底，则由等腰性质及椭圆定义推导，通常不符合此面积特定值。

答案： B

易错点提示： 1. 图形多解性：等腰三角形的腰和底不确定时需分类讨论。
2. 面积公式：牢记焦点三角形面积公式 $S = b^2 \tan\frac{\theta}{2}$（$\theta$ 为顶角）在选择题中可秒杀。
